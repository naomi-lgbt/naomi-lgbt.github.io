Verification Bot

This bot serves as a verification system for new members (to help deter raids and spam). When you add the bot, you can set up a question that new members have to answer correctly to gain access - the bot will assign them a role (which you'll need to set permissions for to serve as a gate system).

Invite to Server

Inspiration Bot

This bot allows you to specify a channel and time - every day at the specified time, the bot will post an inspirational quote in the channel you chose.

Invite to Server

Message Counter

This is a simple bot which counts the number of messages people send in your server. This is helpful when paired with Becca, as it allows you to identify active folks and grant bonus experience points, for example.
